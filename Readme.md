# Tugas 1 - Menghitung Luas (Javascript)
Nama : Jayas Permana  
Mata Kuliah : [Javascript]  
Deskripsi: File `tugas1_luas.html` berisi program JavaScript untuk menghitung luas:
- Persegi panjang (panjang × lebar)
- Bujur sangkar (sisi × sisi)
- Segitiga (½ × alas × tinggi)

Petunjuk:
1. Buka `tugas1_luas.html` di browser (double click atau Open with browser).
2. Isi nilai pada input lalu klik "Hitung".
3. Hasil akan muncul pada bagian "Hasil".
